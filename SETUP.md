# Najd AI - نجد AI
# منصة ذكية لبناء التطبيقات والأنظمة والمواقع والألعاب

## البدء السريع

```bash
flutter pub get
flutter run
```

## الميزات الرئيسية

- ✅ ذكاء اصطناعي متقدم (Gemini + OpenRouter)
- ✅ بناء متعدد المنصات (Android + iOS)
- ✅ دخول آمن (Google + Phone + Guest)
- ✅ مجلس دردشة تفاعلي
- ✅ بناء تلقائي عبر GitHub Actions

## الملفات المهمة

- `lib/main.dart` - التطبيق الرئيسي
- `.env.example` - متغيرات البيئة
- `pubspec.yaml` - المكتبات المستخدمة

## الإعدادات المطلوبة

### GitHub Secrets

أضف هذه المتغيرات في `Settings > Secrets and variables > Actions`:

```
FIREBASE_API_KEY=your_key_here
OPENROUTER_API_KEY=your_key_here
GEMINI_API_KEY=your_key_here
GH_TOKEN=your_token_here
```
